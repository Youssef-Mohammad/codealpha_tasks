# 🎯 Object Detection and Tracking

A real-time computer vision system that detects and tracks objects using a webcam.

This project was developed as part of my internship at **CodeAlpha** to practice real-time AI applications.

---

## 🚀 Features

- Real-time object detection
- Object tracking across frames
- Unique tracking IDs for each object
- Bounding boxes and labels
- Webcam video processing

---

## 🛠 Technologies Used

- Python
- OpenCV
- YOLO (object detection)
- SORT (object tracking)
- NumPy

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/object-detection-tracking.git
```

Navigate to the project directory:

```bash
cd codealpha_object_detection_and_tracking
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
