# 🤖 FAQ Chatbot

A simple chatbot that answers frequently asked questions by finding the most similar question in a predefined FAQ dataset.

This project was built during my internship at **CodeAlpha** to practice Natural Language Processing techniques.

---

## 🚀 Features

- User-friendly chat interface
- FAQ knowledge base
- Text preprocessing and cleaning
- TF-IDF vectorization
- Cosine similarity for question matching

---

## 🛠 Technologies Used

- Python
- Tkinter
- Scikit-learn
- NLTK
- JSON

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Youssef-Mohammad/codealpha_tasks/tree/main/codealpha_chatbot_for_FAQs
```

Navigate to the project directory:

```bash
cd codealpha_chatbot_for_FAQs
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the chatbot:

```bash
python app.py
```
